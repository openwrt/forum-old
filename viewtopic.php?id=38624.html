Here is update for required packages for LUCI

First copy packages with scp

scp \
liblua_* \
libiwinfo_* \
libiwinfo-lua_* \
luci-proto-ppp_* \
lua_* \
luci-proto-core_* \
luci-app-firewall_* \
luci-theme-base_* \
luci-theme-bootstrap_* \
luci-lib-ipkg_* \
luci-i18n-english_* \
luci-sgi-cgi_* \
libubus-lua_* \
libuci-lua_* \
lua_* \
luci-lib-core_* \
luci-lib-nixio_* \
luci-lib-sys_* \
luci-lib-web_* \
luci-lib-core_* \
luci-mod-admin-core_* \
luci-mod-admin-full_* \
uhttpd_* \
uhttpd-mod-ubus_* \
luci_* root@192.168.1.1:/tmp

then install it in this order

opkg install liblua_*.ipk libiwinfo_*.ipk libiwinfo-lua_*.ipk luci-proto-ppp_*.ipk lua_*.ipk luci-proto-core_*.ipk luci-app-firewall_*.ipk luci-theme-base_*.ipk luci-theme-bootstrap_*.ipk luci-lib-ipkg_*.ipk luci-i18n-english_*.ipk luci-sgi-cgi_*.ipk libubus-lua_*.ipk libuci-lua_*.ipk lua_*.ipk luci-lib-core_*.ipk luci-lib-nixio_*.ipk luci-lib-sys_*.ipk luci-lib-web_*.ipk luci-lib-core_*.ipk luci-mod-admin-core_*.ipk luci-mod-admin-full_*.ipk uhttpd_*.ipk uhttpd-mod-ubus_*.ipk luci_*.ipk

And at end just enable and start uhttpd

/etc/init.d/uhttpd enable
/etc/init.d/uhttpd start
