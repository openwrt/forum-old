Hello everybody,

I'm currently facing a trouble with my WDR4300. When I'm connecting to my wlan, the connection is up for some minutes with the client (ubuntu computer) then it disconnects randomly.

The signal is still available but it is not possible for me to re-connect anymore -> I need to restart the router, or i need to restart the computer.

I tested wired and everything is working fine.

I tried with a computer running windows, the signal is excellent (99%) but i cannot connect to the internet anymore, the connection is up for some minutes then it is lost until I restart.

I think the trouble comes from the router.

I tried to set the power of the signal to its maximum : 20db, channel 1. Nothing better.

I'm running Attitude Adjustment 12.09 on WDR4300 v1.7.

You should take the netmask and gateway from the settings of the router. The defaults are 255.255.255.0 and IP of router.

On the router the syslog buffer usually discards the oldest events first, and always contains the newest. You could try increasing it, but I suspect it's more likely there are either no events to report or your time is set wrong on the router.
