ettercap + its dependancies

backported from buildroot-ng to whiterussian

http://openwrt.vcp-springe.de/experimental/ettercap

(just download all the ipk's - using the imagebuilder is the only real sensible option)

--

Also if anyone knows how a can generate a Package file for this (then I would quite like to know)

(In debian I can just do dpkg-scanpackages . /dev/null|gzip -9>Packages.gz) - ipkg-make-index . > Packages (that seems to be the way)

--

I forgot to upload some of the files - I have tested it with http://downloads.openwrt.org/people/nbd … uashfs.trx

(Its too big with its dependancies to fit on the standard squashfs image - if you want to use the micro image do ipkg update; ipkg install zlib; ipkg install libpthread - then download all the files into /tmp (with wget for example) install them one by one with ipkg install whatever.ipk)

Best to use the imagebuilder - download all the files (except openssl-util - which is not needed)
You also need libpthread and zlib (from whiterussian) - put them into your own image (fits fine)
