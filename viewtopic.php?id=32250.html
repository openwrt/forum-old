hi goatroper,

I just uploaded to github the ubnt2-pico2.bin file that was generated when I built the firmware posted earlier in this thread
you can grab it here:

https://github.com/downloads/guinness1/ … r-2011.tgz


I have not tested the binary file on any devices, as I have no Ubiquiti gear.  all my work was done on om1p devices donated to me to assist a mesh project.  if the Ubiquiti device you are using has a single lan/wired port, AR23xx wifi and ~32MB (probably less could work), the configurations in the firmware should be good - disclaimer: just guessing of course

my devices work with the flash program and have no tftpd running on boot.  if your devices operate with a tftp server only, that is probably why the flash app was nonresponsive... more guessing here...

I would be curious to see if it works on Nanostation 2.  would be great to have some feedback

note:  the ubnt2-pico2.bin is the only 'binary' file generated from my build setup.
