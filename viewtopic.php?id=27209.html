sriracha wrote:

The OP's question was "is it officially not possible for me to put openwrt on [a wndr3300]?".  I'm also interested in the answer to that question.

Hey sriracha, JerkZ here.  To answer your question, the WNDR3300 isn't officially supported by OpenWrt... but that doesn't mean you can't put OpenWrt on it.  This is a Broadcom device with a Broadcom N radio.  As in typical Broadcom fashion, the drivers are proprietary (i.e. not open source, so we can't build it).  In this case the binary driver is only available for the 2.4 kernel.  So you should be able to run OpenWrt with Broadcom 2.4 (brcm-2.4 builds).  However, Broadcom recently released the source for some of the wireless N drivers, and therefore we have a new Broadcom driver for the 2.6 kernel in OpenWrt.  So the WNDR3300 may work fine with the 2.6 kernel now (brcm47xx builds)... but I'm not sure.

If you're so inclined, do some testing (at your own risk) with the 2.4 and 2.6 kernels on your WNDR3300.  ASAIK, it's possible, but untested.

See here for more info:
http://wiki.openwrt.org/oldwiki/openwrt … r/wndr3300
