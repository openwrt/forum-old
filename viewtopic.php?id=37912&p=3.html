just in case somebody is interested, here is a short how-to to deploy asterisk over this router

prerequisites:

1. prepare usb-stick with 2 primary partitions for swap (64MB) and for overlay (rest) ext3
2. install latest squashfs firmware

configuring router with external swap and overlay

1. connect router to internet via wan or wifi
2. opkg update
3. opkg install kmod-usb-storage block-mount kmod-fs-ext4
4. reboot
5. mount /dev/sda2 /mnt/sda2
6. tar -C /overlay -cvf - . | tar -C /mnt/sda2 -xf -
7. /etc/init.d/fstab enable
8. update /etc/config/fstab to mount swap partition /dev/sda1
9. update /etc/config/fstab to mount /overlay to /dev/sda2 
10. reboot

installing asterisk18:

1. opkg update
2. opkg install asterisk18
3. /etc/init.d/asterisk enable

Note to self: To enter the OpenWrt failsafe mode, plug the network cable in the LAN/WAN jack, NOT the LAN one.

The rest is standard: Power the router, wait for the blue LED to light up, then press the Reset button once. The LED then blinks very fast and you can setup your PC with static IP address 192.168.1.2 and telnet to 192.168.1.1. No DHCP server is running in failsafe mode.

For those who are interested, I used Ncat portable (for windows), with:

type [specified firmware].bin | ncat.exe -l 3333

and had a second telnet window open to check the file transfer was completed (via filesize). Next, I checked the md5sum matched before writing it to flash with sysupgrade.
