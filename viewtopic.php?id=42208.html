Post #10
nozombian
8 Feb 2013, 22:52

If you don't know the password, you should reset the router first. This is done by quickly pressing and releasing reset button during startup. After a while, one of the leds starts flashing quickly (now is router in failsafe mode with ip address 192.168.1.1), you will have to set your pc manually to use it 192.168.1.2 and then you can telnet to the router using tool putty (you will have to use radio button to switch to telnet in putty, ssh is checked by default). Telnet user name is root, no password needed in failsafe. You will have to write command firstboot (which will erase current config) and then using command passwd you can set your password for user root, which will also disable telnet. Then turn router off and on and it should be in a clean state, ready to flash new firmware, or for your config. Finaly you can use putty to ssh to your router.
