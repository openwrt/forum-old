Hello everyone,

My subject might not be clear because I couldn't find a shorter way to express what I need. Please fully read my question before addressing some solutions.

Currently, I own a D-Link DSL2520U with standard firmware, a Philips SNA6500 with standard firmware and a TP-Link WA-701ND with OpenWrt Attitude Adjustment (12.09 I think). DSL2520 is connected to ADSL2+ line, Philips SNA6500 is connected to its only ethernet port to serve as a switch and a wireless distributor (lets call it WLAN1). Then my WA-701ND is connected wirelessly to SNA6500 in order to distribute wireless signals further in my home, with a different ESSID (lets call it WLAN2)

The problem I am currently having is when I connect to WLAN2, I can connect to internet but I can't connect to devices WLAN1 or LAN. However, I can connect to these devices from WLAN1 or from DSL2520U's USB port. So I have narrowed the problem down to the Access Point containing OpenWrt firmware and that's why I am writing this thing here.

Similarly, I cannot access WA701ND's settings page from WLAN1.

What is the exact setting that I need to adjust so that I can connect to every device in my Local Network regardless the network I am connected? (Locally of course)

Things I have tried:
Setting a static route to 192.168.1.1 (address of DSL-2520U) inside OpenWRT
Setting all rules of firewall as "ACCEPT" regardless zones and also default rules.
Stopping the firewall service (WLAN2's internet connection was disabled after this so I re-enabled it)

PS: All this hassle is for not buying a better, powerful router. They are really expensive here in Turkey due to the currency. I hope that you guys can help a poor student to create up a better network in his house.

It is always best to start with a diagram of what you are doing. As I read it, it looks like you are doing this:

Internet --> DSL2520 --ethernet--> SNA6500 ---z wlan1 zzz--->WA-701ND----zz wlan2 zzz---> windows host2

If NAT is turned on (you are using a wireless Client mode on the WA-701ND then it will BLOCK your windows file sharing on the left side of the network. You may not be able to disable it, and still use your wireless client mode to connect to SNA6500.

Is it possible to connect WA-701ND via wired ethernet? If this is possible, then you would have a "flat" network, and everything should be fine. If not, you have challenges, as you have already figured out.

HTH
