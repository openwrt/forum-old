Hi... Trying to find a router that would fit my needs but there are quite a few so thought i post a question..

Requirements:
- "Powerful" CPU (something like the AR9344 with the 74k MIPS core). Could also be a possibility with a less powerful cpu but with HW support for blowfish. (DSP implementation or something?)
- Detachable antennas.
- 64Mb of ram, the more the better
- 8Mb flash is ok, but would be nice with 16Mb or more.
- 2.4/5Ghz dualband.
- TX-Power, would be good to support atleast 25dBm on 2.4 and 30dBm for 5Ghz.. Would like a minimum of 23/27dBm.

Thinking about TL-WDR4300 but it still has some way to go and would be nice to be able to pick something off the shelf that is already working.. Also waiting for a reply if the tx-power limit is on all devices after version 1.1.

Hope someone is feeling helpful today

I am not sure my following suggestion will be a good option or not, but it certainly is doable.

It is possible to build own WiFi router using any inexpensive & supported ARM platform devices, i.e. a pogoplug, + a USB/RJ45/WiFi dongle + a 4-port 10/100/1000Mbps switch. I recently bought some Seagate GoFLEX NET from Fry's Electronics stores for $9.99/each. It is an ARMv5 architecture clocked @1.2GHz, 128/256 MB RAM/Flash, 1 USB2 port, and 2 SATA ports. Currently, J&R has some sales on some PogoPlug devices for as low as $19.99 + FREE S/H to US).
