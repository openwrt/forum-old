There is a firmware to download and flash and install. However, it is a snapshot.

Just click downloads then goto the snapshots

http://downloads.openwrt.org/snapshots/trunk/ar71xx/

Firmware name:
openwrt-ar71xx-generic-tl-wr1043nd-v1-squashfs-factory.bin
openwrt-ar71xx-generic-tl-wr2543n-v1-jffs2-sysupgrade.bin

At the present time of posting snapshot is r30639 which is higher than r29680.

Please note, releases are stable, and snapshots can be problematic for some people.
No LuCI is an example.

For my understanding and for others as well:
1. Download http://downloads.openwrt.org/snapshots/ … actory.bin
2. Install the firmware via the original webinterface of the router
3. It should come up with the ip 192.168.1.1/24
4. For USB-Support: opkg update, opkg install kmod-usb2, insmod ehci-hcd, insmod usbcore, insmod ehci-hcd
5. Install LUCI: opkg install luci-ssl and enable/start it /etc/init.d/uhttpd enable , /etc/init.d/uhttpd start

Post #23
slh
5 Jul 2012, 00:45

That is a hardware limitation, independent of the firmware.

The TP-Link TL-WR2543ND only has one physical wlan card, to use both bands concurrently you would need a router with two wlan cards (802.11abgn_simultan, like the TP-Link TL-WDR4300).
