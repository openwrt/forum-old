First of all, you need to read Openwrt documentation.
It is a bit long, but good for you to understand what is it.

Then, the ordinary Openwrt firmware you downloaded from openwrt site are NOT include webui,
means that you will never and cannot and impossible to connect in browser after "fresh" flashed.

If you need webui, need to connect in browser,
you need to first "telnet" to 192.168.1.1...... then setup the network part of your openwrted router, then install the webui, start the http service,
afterward you can connect in browser.

You don't need to search the internet,
just go to http://openwrt.org
