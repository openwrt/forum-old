Why do you think you've successfully entered failsafe if telnet doesn't work? If failsafe is working, telnet is enabled. No password is required (via telnet) in failsafe. I believe ping should also work. So to me, it doesn't sound like you've gotten into failsafe.

Other things to try:
- Make sure you set a static IP on your PC in the same range.
- Make sure you are using a wired connection (wireless is not enabled in failsafe)
- A few routers require that you connect to the WAN port instead of a LAN port, but I don't think this is one of them.

Can you also provide a link to the exact file you flashed?
