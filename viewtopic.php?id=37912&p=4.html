How i fix my router tp-link wrn720n V3 Chinese Version:
1.I use custom openwrt firmware "Madox USBIP 36503_2013-05-01 / LuCI Trunk (svn-r9812)"

Copy firmware (file:///home/user/openwrt_TLWR720_usbip_2013-05-01_squash-factory.bin
) to my home folder "user"
 

2.(open terminal and write this)
telnet 192.168.1.1

3.(if connect to router  ok, set new password)
passwd

4.(open other terminal window and write this.Input my password)

scp openwrt_TLWR720_usbip_2013-05-01_squash-factory.bin root@192.168.1.1:/tmp

5.(open first terminal window with telnet and write this)

sysupgrade -v /tmp/openwrt_TLWR720_usbip_2013-05-01_squash-factory.bin

6.all done, after router reboot you can go to 192.168.1.1 and login to openwrt web interface
