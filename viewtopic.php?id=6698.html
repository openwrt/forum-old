Okay, after quite a bit more experimenting, I ultimately managed to make things work out. It turned out that the network card would *only* accept WPA2-PSK with AES for authentication -- otherwise the key would simply not be set. I added some of the experiences made to http://wiki.openwrt.org/WirelessBridgeWithWPAHowto, section 1.7.2 -- hope this will help others to avoid the same trouble.
