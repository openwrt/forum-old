I have installed OpenWrt on this router, but it was already some time ago. I don't remember the exact commands, but here's what I did:
* Get into administration mode
* Backup flash (see wiki page)
* Erase whole flash (area 5, can be risky if you're not willing to solder)
* Upload U-Boot for ARV7518PW (http://downloads.openwrt.org/snapshots/ â€¦ u-boot.img) from trunk to area 5 using 'sx'
* Reboot
* Use tftp in U-Boot to install OpenWrt to flash (at 0xb0040000) and to restore the WLAN EEPROM (last 64k from the backup) to the end of the flash (0xb07f0000).
