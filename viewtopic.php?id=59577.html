The Firmware Image I uploaded contains incorrect Checksum that's why you are unable to Upload the BIN file through the Web Interface of TP LINK Official Firmware.

Follow Below Listed Steps to Upgrade to OpenWRT

1. Change your Computer Ethernet Interface IP Address to "192.168.0.66" (its compulsary else it will not work)
2. Download TFTP Server on your Computer from Here http://tftpd32.jounin.net/download/Tftp … -setup.exe
3. Download OpenWRT Firmware for WR841n v10 from here http://www.megafileupload.com/dyhP/OpenWRT-WR841v10.bin
4. Rename the Downloaded OpenWRT Firmware Filename to "wr841nv10_tp_recovery.bin"
5. In TFTPd32 Software Set your "Current Directory" path to same Folder Location where you have saved "wr841nv10_tp_recovery.bin"
6. Now Plug the CAT5 Cable Connected to Your Computer to Port 1 of TP LINK WR841n v10
7. Now Keep Holding the "WPS/Reset" Button on Back of your Router and at same time Press the Switch ON Button Now wait for 10 Seconds and Release the "WPS/Reset" Button.
8. Your Router is Now being Upgraded to OpenWRT Firmware and in about a Minute when Successfully Upgraded You will see 3 Lights in your Router Power, Port 1 LED & Lock LED.
9. Now change your Computer IP Address to "192.168.1.66" and You can now access the OpenWRT as Usual Open the Command Prompt on Windows and Type "telnet 192.168.1.1" and Now OpenWRT Screen would come up and you can execute any commands as usual and refer to the Official OpenWRT Wiki/Documentation regarding the configurations You can also access the OpenWRT Luci Web Interface at http://192.168.1.1

Hope this helps, This is working perfectly I am using OpenWRT on TL-WR841n v10
