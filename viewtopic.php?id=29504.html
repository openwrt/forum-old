first off all, it's call openwrt, not wrtopen

now to your question: how to edit this file?

you have to connect to the commandline-interface of the router.

Did you even click on the link orca posted?
you have come to step 3 so far, next step is step 4: connect
you will find the info how to connect to the CLI there. if you have set a password already, you can connect over ssh. if not, follow the "first login"-instructions.

on the CLI you can use the editor "vi" to edit your configfiles 

vi /etc/wifidog.conf

... and please don't ask how to use "vi" here, there are tons of information about "vi" on the net.

and your openwrt-router is a linux-based computer so the "normal" wifidog documentation should apply to it too.

eleon216
8 Jun 2011, 14:11

using telnet is a really bad advise, especially if you are planning to run a hotspot!
use telnet the first time, to set a password for the root-user and then connect through ssh (secure shell). if you are on windows get the program "putty" for ssh-connections

telnet without setting a password would mean everybody could connect to the CLI of your router, and would have superuser-rights. this is really a very very bad idea! And if you set a password, the telnet-daemon is disabled per default, so you have to use ssh and you really should.
