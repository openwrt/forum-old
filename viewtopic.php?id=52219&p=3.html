OK, I am using SSH, but with telnet I finish a lot of work in lan network, remote.

Looks like the telnet client was disabled at the same time as the server daemon was disabled in Busybox defaults.
https://dev.openwrt.org/changeset/46809

If you need telnet (from busybox), you need to modify busybox settings and re-compile the firmware.
