*UPDATE* solved

ok, i wasn't reading the wiki properly.

The flashing light i got was some kernel panic probably. it was occurring after the router secured the 192.168.1.1 IP while trying to do the regular startup sequence. it flashed like hell. really fast.

The proper way to go to 'rescue mode' is to press the reset button before pluging in power on the modem. after i did that, the light flashes really slowly. and the router secure the 192.168.0.1 ip

now i can see the firmware upgrade system page. whew.

AFAIK, you should never upgrade firmware this way... Instead, you should download new firmware image file and use `sysupgrade` command-line utility (or upgrade through luci web interface if you prefer GUI).

For now you probably can restore you router following these steps:
1. Unplug power cable;
2. Hold "reset" button at the back side of the device;
3. Plug power cable holding "reset";
4. Wait until status LED begins slowly blinking orange and release "reset" button;
5. Open http://192.168.0.1 or http://192.168.1.1 address with Internet Explorer.
There you'll see original D-Link firmware recovery tool prompt. You will be able to load either original D-Link firmware or OpenWrt factory firmware images using this page. All previous settings will be lost, indeed.
