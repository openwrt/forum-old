Since the ipk will most likely be installed on a different architecture, you can not convert a .deb to a .ipk. You'll need to make an OpenWrt Makefile which will download the source code, cross compile it, and then create an ipk with all the necessary information and files, much like the .deb archive.

A good place to start is by downloading the OpenWrt SVN buildroot and take a look in package/ to get a feel for how it works.



edit: it sounds a bit complicated, but it's actually pretty simple.
